A premium white-palette static marketing website for SchoolOps AI — Education Receivables & Fee Recovery OS.

## What is included

- Full multi-page website
- Home, Product, Connect, Workflow, Dashboard, Pilot, Pricing, About, Contact
- Premium SaaS-style CSS
- Responsive layout
- Dashboard mockup components
- FAQ accordion
- Demo booking form UI
- No external dependencies

## How to use

Open `index.html` in your browser.

You can deploy this as a static site on:

- Vercel
- Netlify
- GitHub Pages
- Any normal web hosting
